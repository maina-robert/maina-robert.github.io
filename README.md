A modern, responsive portfolio website for Robert Maina – Data Analyst & Economics Graduate based in Stuttgart, Germany.

## 🚀 Features
- Dark mode design with animated hero section
- Interactive Chart.js dashboards (skills radar, KPI cards, career timeline)
- Animated skill bars and counters
- Multi-language CV (English/German)
- Responsive for all devices
- Contact form with mailto integration

## 📁 Files
- `index.html` – Main page
- `style.css` – All styles
- `app.js` – Charts, animations, interactions
- `assets/profile.jpg` – Add your profile photo here

## 🌐 Deploy to GitHub Pages
1. Create a GitHub repo named `[username].github.io`
2. Upload all files from this folder
3. Go to Settings → Pages → Branch: main → Save
4. Your site goes live at `https://[username].github.io`
